# Small Business Cybersecurity Guide

Basic cybersecurity practices for small organisations.

## 1. Use Strong Passwords
Passwords should be long and unique.

## 2. Enable Multi-Factor Authentication
This adds an extra layer of protection to accounts.

## 3. Regular Backups
Critical data should be backed up regularly.

## 4. Software Updates
Keep operating systems and applications updated.

## 5. Phishing Awareness
Employees should be trained to identify suspicious emails.

## 6. Access Control
Limit access to sensitive systems and data.
