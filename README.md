# Cybersecurity Risk Assessment Toolkit

This project provides a simple cybersecurity risk assessment toolkit designed for small organisations and charities with limited technical expertise.

The toolkit helps organisations identify cybersecurity risks, assess vulnerabilities, and implement basic security controls.

## Objectives

- Identify common cybersecurity threats
- Assess organisational vulnerabilities
- Provide a structured risk register
- Improve incident response preparedness

## Toolkit Components

1. Risk Register Template
2. Vulnerability Assessment Checklist
3. Incident Response Plan
4. Cybersecurity Awareness Guide

## Use Case

This toolkit can be used by small businesses, charities, and community organisations to perform basic cybersecurity risk assessments and improve their security posture.

## How to Use This Toolkit

1. Use the Vulnerability Checklist to identify potential cybersecurity weaknesses.
2. Document identified risks in the Risk Register Template.
3. Assess likelihood and impact of each risk.
4. Implement mitigation actions.
5. Follow the Incident Response Plan if a cybersecurity incident occurs.
